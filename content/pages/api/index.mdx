# API Reference

Complete reference for Muni's configuration and API options.

## Overview

Muni provides several configuration options and APIs to customize your documentation site. This section covers all available options and how to use them.

## Configuration Files

### next.config.mjs

The main configuration file for your Muni site.

### muniConfig

The navigation and site configuration object.
